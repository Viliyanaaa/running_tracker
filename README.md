# running_tracker
A desktop app for tracking your running progress, built with Python, SQLite and CustomTkinter.

Features
Log runs with date, distance, duration and notes
Auto-calculated pace (min/km);
Dashboard with key stats – total runs, total km, average pace, best pace;
Best times for 5km, 10km, half marathon and marathon;
Best month by total distance;
Interactive distance chart over time;
Delete runs with one click;

Tech Stack:

Python – core logic;
SQLite – local database;
CustomTkinter – UI;
Matplotlib – distance chart;

Getting Started:

Install dependencies:

pip install customtkinter 

Run the app:

python main.py
